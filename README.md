# sauce-generator
## !! don't run it on cmd or else the msg doesn't show

## run it on git bash !!
#### run command:
```
cargo run -- -m <your message> -s <sauce capacity>
```
#### Or the release version:
```
cargo run --release -- -m <your message> -s <sauce capacity>
```
#### Or run the example with command:
```
cargo run
```
